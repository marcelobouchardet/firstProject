Welcome to GitHub Pages.

This automatic page generator is the easiest way to create beautiful pages for all of your projects. Author your page content here using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new branch:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Se você estiver usando o GitHub para Mac, basta sincronizar o repositório e você verá o novo ramo.

Designer Templates

Temos trabalhado alguns modelos bonitos para você usar. Vá em frente e continuar a layouts para navegar por elas. Você pode facilmente voltar para editar sua página antes de publicar. Após a publicação de sua página, você pode revisitar o gerador de página e mudar para outro tema. O conteúdo da página será preservada se permanecesse formato de desconto.

Rather Drive Stick?

Se você preferir não usar o gerador automático, aperte um ramo chamado gh-páginas em seu repositório para criar uma página manualmente. Além de apoiar o conteúdo HTML regular, Pages GitHub apoiar Jekyll, um blog ciente site estático gerador simples, escrito por nosso próprio Tom Preston-Werner. Jekyll torna mais fácil para criar cabeçalhos e rodapés em todo o site, sem ter que copiá-los através de cada página. Ele também oferece suporte blogue inteligente e outros recursos avançados de templates.

Authors and Contributors

You can @mention a GitHub username to generate a link to their profile. The resulting <a> element will link to the contributor's GitHub Profile. For example: In 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett), and Tom Preston-Werner (@mojombo) founded GitHub.
